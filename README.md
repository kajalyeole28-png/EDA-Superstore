# EDA-Superstore
Exploratory Data Analysis on Superstore Sales Dataset using Python. 

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Superstore dataset using Python.  
The objective is to analyze sales trends, customer segments, product performance, and profitability to generate business insights.

## Business Objectives
Analyze overall sales performance
Identify profitable and loss-making categories
Compare regional performance
Study discount impact on profit
Detect trends over time

## Tech Stack
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

## Dataset Details
The dataset includes:
Order Date
Ship Date
Customer Segment
Region
Category & Sub-Category
Sales
Profit
Discount
Quantity

## Exploratory Analysis Performed
Data Cleaning
Missing value handling
Duplicate check
Univariate Analysis
Bivariate Analysis
Correlation Heatmap
Sales & Profit Trend Analysis
Regional & Segment Comparison
